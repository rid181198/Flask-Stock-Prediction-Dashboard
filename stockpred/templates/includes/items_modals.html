Meta data of the inputs

Basic inputs

  • Code: Every company has a specific code to obtain the stock data. Please refer this link
  • Cancel models: This will remove user-given models for both comparison and long term prediction.
  • Long prediction: Need to enter the number of days you want to predict (limited to one year). The prediction will be based on the default model.

Change the model

  • Neurons More neurons will make model complex and time consuming but the predictions would be accurate. Putting extreme values wouldn't make any difference. Please refer this link
  • Epochs Epochs will run the same model for the number you have given. Again extreme values will not affect the results because of saturation
  • Lookback This will create a history of stock data. Model learns from this history. For long term prediction, the more the lookback time, the more accurate the results would be. Please refer this link
  • Loss/Cost function Model will try to minimize the errors based on this function. You hae choices of MSE (mean squared error), MAE, MAPE, and MSLE. Please refer this link
  • Optimizer This will optimize the model and make it faster using Adam, SGD (stochastic gradient descent), and RMSprop functions. Please refer this link
